Pet Shop Management Website
This is a fully functional Pet Shop Management Website developed using the MERN stack (MongoDB, Express, React, Node.js). It allows users to sign up, view products and pets, place orders, view services, and contact the shop. The admin panel offers full management capabilities, including product and pet management, order management, and admin details management.

Features
User Features:
Signup: Users can sign up and create an account to place orders.
View Products and Pets: Users can browse the available pets and products.
Order Products and Pets: Users can add pets or products to their cart and place orders.
View Services: Users can view available services provided by the pet shop.
Contact Us: Users can send messages to the pet shop or use the contact form.

Admin Features:
Manage Products and Pets: Admins can add, update, or delete products and pets.
Manage Customer Orders: Admins can view customer orders, mark them as delivered, and send receipts.
Admin Details Management: Admins can update their personal information and change login details.
Download Receipts: Admins can download receipts to send to customers after delivering orders.

Tech Stack
MongoDB: Database for storing user, product, and order data.
Express.js: Backend framework for building RESTful APIs.
React.js: Frontend library for building the user interface.
Node.js: JavaScript runtime environment to run the backend.
JWT (JSON Web Token): Used for secure user authentication and authorization.
Bootstrap: Frontend styling framework.

Installation and Setup
Prerequisites
Ensure you have the following software installed:

Node.js (version 14 or above)
MongoDB (if using locally) or MongoDB Atlas for cloud database

Running the Project
To run the application locally, follow these steps:

Start the backend server:
Go to the server directory and run:
npm start

Start the frontend development server:

Go to the client directory and run:
npm run dev
